## 一、项目管理（欢迎页）

| # | 测试功能 | 操作步骤 | 验收标准 |

## 二、设置模态框

### 2.1 API 配置

| # | 测试功能 | 操作步骤 | 验收标准 |
|---|---------|---------|---------|
| 2.1.1 | 填写 API Key | 选择厂商 → 输入 Key → 保存 | 底部状态栏显示"已连接"（绿点） |
| 2.1.2 | 测试连接 | 填写 Key → 点击"测试" | 显示"连接成功"或具体错误信息 |
| 2.1.3 | 切换厂商 | 选择不同厂商 Tab | Base URL 自动切换为该厂商默认值 |

| 2.1.6 | 未配置 Key 时使用 AI | 不填 Key → 尝试 AI 对话 | 提示"未配置 API Key"，不崩溃 |
